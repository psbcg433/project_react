## 🚀 How to Run

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/react-mini-projects.git
   ```

2. Navigate into a specific project:
   ```bash
   cd 01_01_MUI_SocialMediaFeed
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the app:
   ```bash
   npm start
   ```
